平成26年度 秋季都大会 WEEK3
2014年09月21日 10:00
at 早実グラウンド

1Q 2Q 3Q 4Q
都立西高 OWLS 0 14 7 7 28
早稲田実業学校 WARRIORS 0 0 0 0 0

ゲームレポート

第1クオーター

西高キックリターンで試合開始。 序盤は双方探り合いの様相。 西高オフェンスはQB#15からのパスがインターセプトされるなど波に乗り切れないが、 ディフェンスがしっかり守り切る。

第2クオーター

西高自陣10ydsからのオフェンスはRB#42、24、22、21が交互にRUNプレーを展開。 最後は#21のカウンタープレーでTD、先制。 キック成功で7-0。 更に次のオフェンスシリーズでは#42のRUNプレーを中心に敵陣に侵攻。 最期も#42の中央を付くRUNプレーでTD。 キック成功で14-0。 この後のシリーズをディフェンスがシャットアウトし前半終了。 前半は効果的なRUNプレーと安定したディフェンスで危なげない展開。

第3クオーター

後半は西高キックオフで試合再開。 早実のドロープレーなどでゲインを許すも、スクリーンプレーをインターセプトし ターンオーバー。 このシリーズを西高オフェンスが確実に生かし、最後は#21のRUNプレーでTD。 キック成功で21-0。

第4クオーター

後半はQB#15からＴＥ#87、ＷＲ#85へのpassプレーも決まり始める。 ディフェンスはLB#6のQBサックやDL#74の素早いタックルで相手にチャンスを与えない。 オフェンス50yds付近からのドライブで最後は#24の中央へのダイブプレーで追加点。 キック成功で28-0。
